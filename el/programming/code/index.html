Κώδικας

Προγράμματα ανοιχτού κώδικα που έχω αναπτύξει και κώδικας από τα βοηθήματα του παρόντος ιστολογίου.
This page is also available in english.
DuckRails

Ένα βοηθητικό εργαλείο ανάπτυξης κώδικα γραμμένο σε Ruby on Rails.

Ο βασικός σκοπός του είναι να επιτρέπει στους προγραμματιστές να μοκάρουν γρήγορα και δυναμικά API endpoints (διαθέσιμο docker image).

Πατήστε εδώ για οδηγίες εγκατάστασης και χρήσης.

goto

Ένα εργαλείο για το bash που επιτρέπει την γρήγορη καταχώρηση ψευδωνύμων σε φακέλους (directory aliases) και την πλοήγηση σε αυτούς με αυτόματη συμπλήρωση (bash completion).

elman

Ένα script για ευρετηριασμένη αναζήτηση πλήρους κειμένου (lol - εννοώ: full text searching) των Linux βοηθητικών σελίδων (man pages) με χρήση του Elasticsearch. Υλοποιήθηκε για πειραματισμό με την ιδέα που περιγράφεται σε αυτό το άρθρο.

(╯°□°）╯︵ ┻━┻ - table_flipper
Ένα άχρηστο gem που προσθέτει το αναποδογύρισμα του τραπεζιού στην αρχή των μηνυμάτων λαθών της Ruby.
rspec-matchers-controller_filters
Ένα gem για την επαλήθευση εκτέλεσης των φίλτρων που ορίζονται σε ενέργειες των ελεγκτών (controller actions) μιας Ruby on Rails εφαρμογής.
Ruby GTK ToDo Tutorial Application
Ο κώδικας του βοηθήματος για τη δημιουργία μιας απλής GTK+ εφαρμογής (ToDo) με χρήση της Ruby.
Ruby on Rails ToDo Tutorial Application
Ο κώδικας της σειράς βοηθημάτων για τη δημιουργία μιας απλής Ruby on Rails ToDo εφαρμογής.
Hangman Tutorial Application
Ο κώδικας του βοηθήματος για τη δημιουργία του παιχνιδιού "Κρεμάλα" με Ruby on Rails.
release_logs
Επιπλέον λειτουργίκοτητα της εφαρμογής Redmine για τη διαχείριση releases έργων.
LofoCats API
Δείγμα API που αναπτύχθηκε με Ruby on Rails.
LofoCats UI
Δείγμα διαδικτυακής εφαρμογής που καταναλώνει το LofoCats API, υλοποιημένη με Ruby on Rails.
